About SkyNova

SkyNova is an AI‑assisted concept drone designed as a class project to demonstrate web design fundamentals: semantic HTML, responsive CSS, accessibility, performance, and a cohesive design system.

Design Principles

  • Usability: clear navigation, logical hierarchy, readable text.
  • Accessibility: alt text, keyboard focus styles, proper labels.
  • Consistency: shared components and typography across pages.

Inspiration

We took cues from 1950s magazines—bold color blocking, slab headlines, and playful accents—while keeping the code modern and standards‑based.

© SkyNova. Designed with a love for mid‑century modern.
